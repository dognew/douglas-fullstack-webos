This is my official personal and professional website. Moving away from traditional static layouts, I architected this project as an interactive Operating System interface. The goal is to practically demonstrate advanced concepts in software engineering, system architecture, and user experience (UX).

## 🚀 The Vision

The project is designed to mirror the complete OS lifecycle, featuring:

* **Custom Boot Sequence:** A simulated BIOS that processes and displays real-time hardware logic (such as CPU and memory detection).


* **Dynamic Bootloader (GRUB):** Intelligent device detection logic that automatically switches between Desktop (Pro Environment) and Mobile (Touch-optimized UI) modes.


* **System Transitions:** A full boot flow including animated Splash screens and a functional Session/Login Manager.


* **Environment Management:** A multi-window desktop interface built to showcase my portfolio, academic journey, and professional services in an immersive way.



## 🛠️ Technical Stack

* **Backend:** Laravel 11 (PHP 8.3.30) built as a high-performance JSON API, faithfully mirroring my LiteSpeed production environment.


* **Frontend:** React.js with TypeScript, leveraging strict typing for complex state machine management.


* **Infrastructure:** Orchestrated via Docker to ensure parity between local development and final deployment.


* **Database:** MySQL 8.0 with managed volumes to ensure persistent user preferences and settings.



This repository serves as a technical manifesto, showcasing my expertise in containerization, state management, and modern web architectures.
