[🇺🇸 English Version (README.md)](README.md)

Este é o meu site pessoal e profissional oficial. Fugindo do padrão de layouts estáticos tradicionais, projetei esta aplicação como uma interface de Sistema Operacional interativo. O objetivo é demonstrar, na prática, conceitos avançados de engenharia de software, arquitetura de sistemas e experiência do usuário (UX).

## 🚀 A Visão

O projeto foi concebido para espelhar o ciclo de vida completo de um SO, incluindo:

* **Sequência de Boot Customizada:** Uma BIOS simulada que processa e exibe lógica de hardware em tempo real (como detecção de CPU e memória).


* **Bootloader Dinâmico (GRUB):** Implementação de detecção inteligente de dispositivos para alternar automaticamente entre os modos Desktop (Ambiente Pro) e Mobile (Interface otimizada para toque).


* **Transições de Sistema:** Fluxo completo de inicialização com telas de Splash animadas e um Gerenciador de Login funcional.


* **Gerenciamento de Ambiente:** Área de trabalho multi-janelas desenvolvida para apresentar meu portfólio, trajetória acadêmica e serviços profissionais de forma imersiva.



## 🛠️ Stack Técnica

* **Backend:** Laravel 11 (PHP 8.3.30) estruturado como uma API JSON de alta performance, replicando fielmente meu ambiente de produção em LiteSpeed.


* **Frontend:** React.js com TypeScript, utilizando tipagem rigorosa para o gerenciamento de máquinas de estado complexas.


* **Infraestrutura:** Orquestração via Docker para garantir consistência entre o desenvolvimento local e o deploy final.


* **Banco de Dados:** MySQL 8.0 com volumes gerenciados, assegurando a persistência das preferências e configurações do usuário.



Este repositório é mais do que um site; é um laboratório vivo onde aplico conceitos de conteinerização, gerenciamento de estado e arquiteturas modernas de web.
